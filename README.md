## iKuai Bypass

fork 自 https://github.com/ztc1997/ikuai-bypass/

## 主要修改点
- 并发处理 运营商/IP分流 和 域名分流  
- 更新成功后再删除旧规则  
- 支持无docker运行  
- 支持单次运行参数`-r nocron`忽略配置文件的cron配置 `./ikuai-bypass_exe -c config_example.yml -r nocron` v0.1.3以后版本
